# Springer Capital

A modernized, high-performance web application for Springer Capital built with Next.js, Tailwind CSS, and TypeScript.

## Overview

This project represents a complete architectural modernization of the core Springer Capital website, delivering a pixel-perfect implementation using modern engineering standards. It is designed to be fully compliant with international privacy regulations and email service provider requirements.

Key features include:
- **Modern Stack**: Built with the Next.js App Router, React, TypeScript, and Tailwind CSS v4.
- **Compliance & Trust**: Upgraded legal documents (Privacy Policy and Terms of Service) and Cookie Banner management, designed for strict GDPR, CAN-SPAM adherence, and AWS SES production standards.
- **Communications Architecture**: Contact forms featuring transactional controls and explicit marketing opt-ins.

## Getting Started

First, install dependencies:

```bash
npm install
# or yarn install, pnpm install, bun install
```

Then, run the development server:

```bash
npm run dev
# or yarn dev, pnpm dev, bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

* **Core Framework:** Next.js
* **Language:** TypeScript
* **Styling Engine:** Tailwind CSS
